# 🍕 PizzaPrice AI - Preditor de Preços

Este projeto utiliza **Machine Learning** para prever o preço de uma pizza com base no seu diâmetro, utilizando um modelo de **Regressão Linear**.

## 🚀 Tecnologias Utilizadas
- **Python 3.13**
- **Pandas**: Manipulação e análise de dados.
- **Scikit-Learn**: Criação e treinamento do modelo de Inteligência Artificial.
- **Streamlit**: Interface web interativa.
- **Poetry**: Gestão moderna de dependências e ambientes virtuais.

## 📊 O Projeto
O objetivo é demonstrar o fluxo completo de um projeto de Ciência de Dados:
1. **Coleta**: Dados estruturados em CSV.
2. **Treinamento**: Uso de Regressão Linear para encontrar a correlação entre tamanho e preço.
3. **Interface**: Dashboard interativo para predições em tempo real.

## 🛠️ Como rodar o projeto
1. Instale o Poetry: `pip install poetry`
2. Instale as dependências: `poetry install`
3. Inicie o App: `poetry run streamlit run app.py`
